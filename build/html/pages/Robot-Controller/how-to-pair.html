遥控手柄配对指南

该操作指南用于遥控器第一次配对机器人时的步骤指南。

配对步骤

调试线必须插入DBG口，并且数据线A面朝外。
连接终端
ssh root@192.168.55.1

  1. 主控执行以下指令

    crsf-app -bind

    可以观察到返回：

    正在检查串口通讯状态...
uart connect success
正在进入配对模式...
bind mode success

    请打开遥控器，左长按右侧按键进入TOOLS -> ExpressLRS -> [Bind]，手动搜索配对。

  2. 遥控器开机后右边按键向左推进入界面后按键依次进入Tools -> ExpressLRS -> bind模式，进行配对接收机。

  3. 配对完成返回pair success。
